# Medicine Count & Expiry

Track your medicine inventory and expiry dates directly in Home Assistant.

## Features at a glance

- 📋 Track medicines with name, expiry date, quantity, and location
- 🚨 Automatic alerts for expired and expiring-soon medicines
- 🤖 Optional Claude AI label scanning
- 🔍 Search and filter your inventory
- 📊 HA sensor entities for automations
- 🃏 Beautiful Lovelace dashboard card

## Quick Setup

1. Install via HACS
2. Restart HA
3. Go to **Settings → Devices & Services → Add Integration → Medicine Count & Expiry**
4. (Optional) Enter your [Anthropic Claude API key](https://console.anthropic.com) for AI features
5. Add the Lovelace card to your dashboard
